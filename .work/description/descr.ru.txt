Расширение для разработчиков расширений.

Используй клавиатурное сокращение (Alt-Z по умолчанию) или контекстное меню на любой вкладке, чтоб перезапустить активные распакованные расширения.

За:

- Быстрая перезагрузка из контекстного меню на любой вкладке или с помощью клавиатуры (Alt-Z по умолчанию, но можно изменить)
- Перезагрузка только активных (включенных) расширений
- Если Options page открыта она также будет перезагружена
- Возможность также перезагрузить активную вкладку (см. страницу параметров расширения). Полезно при разработке content script расширений
- Возможность отключить перезагрузку из контекстного меню (см. страницу параметров расширения)

Против:

- Перезагрузка таким образом закрывает background debugger window (конечно, если было открыто). В данный момент нет публичного API для перезапуска.

Исходный код - https://github.com/onikienko/dev-extensions-reload/
